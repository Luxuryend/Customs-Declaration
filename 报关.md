# 报关类型
进口报关：涉及进口征税及付汇
出口报关：涉及出口退税及收汇
进境报关：暂缓征税，涉及付汇
出境报关：涉及收汇
进出口填报报关单、进出境填报备案清单

### 进口货物完税价格审定表（按成交地点划分）

| 成交地点/价格术语 | 审定与计算原则 | 完税价格计算公式 |
| :--- | :--- | :--- |
| **我国口岸到岸价 (CIF)** | 货物在我国口岸交货。如属正常交易并经海关审定，直接以到岸价格作为完税价格。 | $$完税价格 = CIF$$ |
| **境外口岸离岸价 (FOB)** | 货物在境外口岸交货。应另加从境外口岸运抵我国口岸前实际支付的运杂费、保险费。 | $$完税价格 = \frac{FOB + 运费}{1 - 保险费率}$$ |
| **离岸加运费价格 (CFR)** | 货物在境外口岸交货，但运费已付。需另外加计从境外到我国口岸的保险费。 | $$完税价格 = \frac{CFR}{1 - 保险费率}$$ |

CIF (Cost, Insurance and Freight)
中文含义： 成本加保险费、运费（俗称“到岸价”）。

FOB (Free On Board)
中文含义： 船上交货（俗称“离岸价”）。
海关审价： 价格中不包含运费和保险费。

CFR (Cost and Freight)
中文含义： 成本加运费（旧称 C&F）。
含义： 卖方负责支付将货物运至目的港所需的运费，但不负责办理保险。
海关审价： 价格中已含运费，但不含保险费。


# 海关业务管理与税费计算复习笔记

---

## 一、 进出境监管体系概览

### 1. 监管对象与期限
- **货物分类**：按贸易管制分为禁止、限制、自由进出口；按流向分为进出口货物、过境/转运/通运货物、暂时进出境货物。
- **监管期限**：
  - **进口**：自进境起至办结海关手续止。
  - **出口**：自向海关申报起至出境止。
  - **过境货物**：自进境起至出境止。

### 2. 法律关系相对人
- **报关人**：进出口货物收发货人、进出境运输工具负责人、进出境物品所有人及代理人。
- **报关活动相关人**：经营保税加工、物流、仓储业务的企业，转关运输承运人等。

---

## 二、 专项监管制度

| 制度名称 | 核心内容与监管要点 |
| :--- | :--- |
| **保税监管** | 货物进境后加工或储存再复运出境；若不能复运出口需办理进口纳税手续。 |
| **法定减免税** | 依照法律规定直接减免，海关一般不进行后续管理。 |
| **特定减免税** | 针对特定地区/用途/企业；监管年限内未经批准不得擅自出售或转让。 |
| **临时减免税** | 根据特殊情况给予的特别临时性减免优惠。 |

---

## 三、 进出口税费计算公式 (核心考点)

### 1. 完税价格审定 (计税底数)
| 类别 | 成交方式 | 计算公式 | 说明 |
| :--- | :--- | :--- | :--- |
| **进口** | **CIF (到岸价)** | $$完税价格 = CIF$$ | 正常成交并经海关审定直接作为完税价格。 |
| **进口** | **FOB (离岸价)** | $$完税价格 = \frac{FOB + 运费}{1 - 保险费率}$$ | 需加计运抵我国口岸前的运杂费、保险费。 |
| **进口** | **CFR (含运费价)** | $$完税价格 = \frac{CFR}{1 - 保险费率}$$ | 价格含运费，需补齐保险费作为完税价格。 |
| **出口** | **实务公式** | $$完税价格 = \frac{FOB}{1 + 出口税率}$$ | 剔除出口关税，得出不含税的完税价格。 |

### 2. 具体税款计算
- **进口关税**：
  $$关税 = 完税价格 \times 关税率$$
- **进口增值税** (计税基数含关税)：
  $$增值税 = (完税价格 + 关税) \times 增值税率$$
- **进口消费税** (从价定率)：
  $$消费税 = \frac{(完税价格 + 关税) \times 消费税率}{1 - 消费税率}$$

---

## 四、 业务流程主要步骤
1. **进出境阶段**：完纳税费，提交许可证件。
2. **海关作业**：审核单证、查验货物、征收税费、签章放行。
3. **结关阶段**：放行后或离境后办结海关手续。
